# Temporal Debt - User Guide

## Welcome to TEMPORAL DEBT
*Time is a loan you cannot afford.*

## Getting Started

### Running the Game
```bash
cd temporal_debt
python main.py
```

## New Player Tutorial

When you start the game, you'll see a **7-step interactive tutorial** that teaches you everything you need to know:

1. **Welcome Screen** - Introduction to the game world
2. **Movement Tutorial** - Learn basic controls  
3. **Time Freeze Mechanic** - Master the core gameplay
4. **Debt System** - Understand the consequences
5. **Time Anchors** - Advanced teleportation technique
6. **Level Goals** - How to complete each level
7. **Ready to Play** - Final preparation

**Press SPACE** to advance through each tutorial step.

## Controls

### Basic Controls
- **WASD** or **Arrow Keys** - Move your character
- **SPACE** (hold) - Freeze time (accumulates debt)
- **ESC** - Pause game

### Advanced Controls
- **Q** - Place Time Anchor (costs debt)
- **E** - Recall to Time Anchor (costs debt)
- **TAB** - Toggle controls/tutorial display

### Pro Tip
Press **TAB** anytime during gameplay to show/hide the controls reference panel!

## Game Mechanics

### Time Freeze
- Hold SPACE to freeze time
- Enemies stop moving
- You can still move freely
- **Warning**: Accumulates temporal debt!

### Temporal Debt
- Debt increases when you freeze time
- Debt increases when using anchors
- Reach bankruptcy (20 seconds) = Game Over
- Use **Debt Sinks** to reduce debt

### Time Anchors
- Press **Q** to place an anchor (costs 2s debt)
- Press **E** to teleport back (costs 1s debt)
- Only one anchor at a time
- Strategic positioning is key!

## Interactive Objects

### Debt Sinks (Blue Crystals)
- Absorb temporal debt
- Limited uses (3 per sink)
- Plan your route carefully
- Essential for completion

### Debt Mirrors (Reflective Panels)
- Charge with time energy
- Reflect as projectiles
- Creative puzzle solutions
- Advanced technique

### Debt Bombs (Red Spheres)
- Trigger explosion zones
- Create time acceleration
- Clear enemies
- High risk, high reward

### Exit Zones (Green Portals)
- Level completion
- Requires reaching safely
- Often guarded
- Strategic approach needed

## Enemy Types

### Patrol Drones (Red)
- Follow predictable patterns
- Horizontal/vertical movement
- Avoidable with observation
- Standard threat

### Seeker Enemies (Orange)
- Track player position
- Intelligent pathfinding
- Cannot be predicted
- Require active evasion

### Temporal Hunters (Purple)
- **Only move during time freeze!**
- Punish freeze overuse
- Static when unfrozen
- Strategic counter

### Debt Shadows (Black)
- Spawn at high debt (15s+)
- Chase relentlessly
- Very dangerous
- **Avoid high debt!**

## Strategy Tips

### For Beginners
1. **Observe First** - Watch enemy patterns before moving
2. **Freeze Sparingly** - Debt accumulates fast
3. **Plan Your Route** - Know where Debt Sinks are
4. **Use Walls** - Hide behind cover
5. **Stay Calm** - Panic causes mistakes

### For Advanced Players
1. **Anchor Positioning** - Place anchors strategically
2. **Debt Management** - Balance risk vs. reward
3. **Enemy Manipulation** - Use freezes to create gaps
4. **Mirror Mastery** - Advanced puzzle solutions
5. **Speed Running** - Minimal debt clears

### Pro Strategies
1. **Debt Cycling** - Use sinks at optimal times
2. **Pattern Memorization** - Learn enemy routes
3. **Anchor Chains** - Multiple anchor placements
4. **Risk Calculation** - When to freeze vs. dodge
5. **Shadow Prevention** - Never exceed 15s debt

### Common Issues

**"I keep going bankrupt!"**
- Use time freeze less
- Find and use Debt Sinks
- Plan your route better
- Avoid Temporal Hunters

**"Enemies are too fast!"**
- Watch patterns first
- Use walls for cover
- Strategic freeze usage
- Time your movements

**"I'm stuck on a level!"**
- Try different paths
- Use all interactables
- Watch for hidden routes
- Take breaks and retry

**"What does this enemy do?"**
- Red = Patrol Drones (predictable)
- Orange = Seekers (chase you)
- Purple = Hunters (move when frozen)
- Black = Shadows (high debt penalty)
